Question No-1

Write a code to insert an image

        To insert an image in html, we use img tag. In this tag we have to use src (source) attribute
        to provide the image url. And other than that we give alt (alternate) attribute, whose value
        will be displayed on the screen if the image url given in source is not correct.

        Code:

        <h1>Image</h1>
        <img src=”link address of the image” alt=”alternate text” />


Question No-2

Create a form contain input field like name age email and contact

        Whenever you are creating in HTML forms, you should create an input field inside that
        we must add attributes like type, name, id and value. So that when you are submitting
        the form you are able to see the output given by the user in the url.

        Code:

        <h1>Create an form</h1>
        <form action="#">
        <label for="name">Enter your name :- </label>
        <input type="text" name="name" id="name" placeholder="Your Name"><br><br>

        <label for="age">Enter your age :- </label>
        <input type="number" name="age" id="age" placeholder="Your Age"><br><br>

        <label for="mail">Enter your Email :- </label>
        <input type="email" name="email" id="mail" placeholder="Your Email"><br><br>

        <label for="phone">Enter your contact :- </label>
        <input type="tel" name="contact" id="phone" placeholder="Your Number"><br><br>

        <button>Submit</button>

        </form>


Question No-3

Create a table and explain how to achieve colspan

        Colspan is used to combine two columns together.

        Code:

        <h1>Example of Colspan</h1>
        <table border="1">
            <tr>
                <th>Cell 1</th>
                <th>Cell 2</th>
                <th>Cell 3</th>
            </tr>
            <tr>
                <td colspan="2">Ball</td>
                <td>Ball</td>
            </tr>
            <tr>
                <td>Doll</td>
                <td>Doll</td>
                <td>Doll</td>
            </tr>
        </table>
